电脑配件销售系统包括前台销售系统及后台管理系统，基于 Spring Boot 及相关技术栈开发。 前台商城系统包含首页门户、商品分类、新品上线、首页轮播、商品推荐、商品搜索、商品展示、购物车、订单结算、订单流程、个人订单管理、会员中心、帮助中心等模块。 后台管理系统包含数据面板、轮播图管理、商品管理、订单管理、会员管理、分类管理、设置等模块。

  • 本项目是太原理工大学软件学院16级某学生毕业设计
  • 本项目的密码采用MD5util加密方式存储到数据库
  • 本项目添加商品的富文本编辑器采用KindEditor插件
  • 本项目中的网格采用jQuery中的jqGrid网格插件
  • 本项目的轮播图采用swiper插件
  • 本项目的登录验证码采用kaptcha插件
  • 本项目的消息提示框采用sweetalert插件
  • 本项目的分页功能采用PageResult插件
  • 本项目的图片上传功能采用AjaxUpload插件
